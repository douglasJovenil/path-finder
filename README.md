# Path Finder

This project implements a dynamically allocated matrix, reads this [file](data/labirinto.txt) that contains a maze and solve it, where:

- 5 is the entry
- 9 is the exit
- 1 is a wall
- 0 is a possible path
- 4 is a accomplished path
